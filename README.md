# striketracker.github.io

This repository generates a GitHub Pages website with the Strike Tracker: https://striketracker.github.io/. 

The html file [index.html](index.html) contains an iframe that loads the Strike Tracker site within the document. The Strike Tracker map and data are based on R Shiny.

The favicon is sourced from [favicon.io](https://favicon.io/emoji-favicons/raised-fist/) under the CC-BY 4.0 license.

The Strike Tracker has been founded by [Alexander Busch](https://www.alexander-busch.eu/), [Jim Frindert](https://berlin-brandenburg.dgb.de/ueber-uns/ansprechpartner/bereich-jugend), [Lukas Lehner](https://lukaslehner.github.io/) and [Kilian Weil](https://www.hertie-school.org/en/who-we-are/profile/person/weil) in 2023.

Citation: Busch, A., J. Frindert, L. Lehner, and K. Weil (2023). Strike Tracker: A Living Dataset of Industrial Actions in Germany.
